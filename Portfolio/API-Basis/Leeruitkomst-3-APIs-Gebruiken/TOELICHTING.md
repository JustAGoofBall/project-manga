# 🔌 Leeruitkomst 3: API's Gebruiken

## Wat Ik Toon
Ik demonstreer dat ik **API's kan consumeren** door:
- Requests van de React frontend naar Express backend op te stellen
- Fetch API / Axios verwendt om data op te halen
- Response data verwerken in frontend components
- Error handling implementeren

**Bewijs:**
- React frontend components die mijn Express API consumeren
- API calls via fetch/axios
- Data binding in components
- Loading states en error states

## Hoe Dit In Mijn Project Zichtbaar Is
- **Frontend:** `/frontend/src/` bevat React components
- **API Calls:** Components gebruiken `fetch()` om naar Express endpoints te gaan
- **Context/Services:** Mogelijk useContext of service-layer voor API management
- **Demonstratie:** Zie [USAGE-EXAMPLES.md](USAGE-EXAMPLES.md) voor code voorbeelden
