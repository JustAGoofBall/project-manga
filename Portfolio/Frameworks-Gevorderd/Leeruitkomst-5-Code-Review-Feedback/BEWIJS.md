### ✅ Bewijs
- Checklist van what to look for in code review
- Best practices

### 💬 Feedback Verwacht Van
- Docent Timo

### 📝 Mijn Uitleg

**MIJN CODE REVIEW CHECKLIST**

**1. Functionaliteit**
- [ ] Code doet wat het moet doen
- [ ] Edge cases handled
- [ ] Error cases handled

**2. Code Quality**
- [ ] Naming is duidelijk
- [ ] Functions zijn klein & focused
- [ ] No code duplication (DRY)

**3. Architecture**
- [ ] Scheiding van concerns
- [ ] Correct design patterns
- [ ] Models/Controllers apart

**4. Security**
- [ ] Input validatie
- [ ] SQL injection prevention
- [ ] Auth/Authorization checks

**5. Performance**
- [ ] Database queries optimized
- [ ] No N+1 queries
