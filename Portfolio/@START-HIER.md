## 🏆 Project: Manga API

Al deze leeruitkomsten demonstreer ik met **1 project**: een RESTful API voor Anime/Characters management

**Tech Stack:**
- **Backend:** Express.js (Node.js)
- **Frontend:** React.js + Vite
- **Database:** MySQL
- **Testing:** Jest + Supertest
- **Authentication:** JWT + Bcrypt

**Featuress:**
- Create, Read, Update, Delete anime
- Manage characters per anime
- User authentication & authorization
- Favorites system
- Rating system
- 60+ automated tests
- Full security implementation

## 📋 Hoe Deze Portfolio Te Gebruiken

### Optie 1: Snelle Scan (15 minuten)
1. Check de folder structure hierboven
2. Read TOELICHTING.md in elk leeruitkomst folder
3. Snap wat ik heb gedaan

### Optie 2: Grondige Review (1 uur)
1. Open elke leeruitkomst
2. Read de format-specifieke file (vragen, checklist, etc)
3. Kijk naar evidence/ voorbeelden
4. Klik naar code in project voor details

### Optie 3: Deep Dive (2+ uur)
1. Do full review per vak
2. Verificeer code quality
3. Run tests: `npm test`
4. Run coverage: `npm run test:coverage`
5. Check frontend: `cd frontend && npm run dev`
