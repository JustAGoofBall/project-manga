# 🔧 Leeruitkomst 3: Framework + ORM CRUD Operaties

## Wat Ik Toon
Ik demonstreer dat ik **middels Express.js framework met MySQL2 (quasi-ORM)** alle CRUD operaties kan uitvoeren:
- **Create** - Nieuwe records toevoegen
- **Read** - Data ophalen (list + detail)
- **Update** - Records aanpassen
- **Delete** - Records verwijderen

**Bewijs:**
- Code voorbeelden van alle CRUD operaties
- Models die database operations encapsuleren
- Controllers die CRUD endpoints exposen
- Werkende API endpoints

## Hoe Dit In Mijn Project Zichtbaar Is
- **Models:** [models/](../../../models/) folder - animeModel, characterModel, userModel, etc
- **Controllers:** [controllers/](../../../controllers/) - Business logic per resource
- **Routes:** [routes/](../../../routes/) - HTTP endpoints voor CRUD
- **Demonstratie:** Zie [CRUD-OPERATIONS.md](CRUD-OPERATIONS.md) voor code voorbeelden
