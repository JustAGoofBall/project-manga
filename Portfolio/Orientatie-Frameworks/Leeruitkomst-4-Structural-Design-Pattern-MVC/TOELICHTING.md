# 🏗️ Leeruitkomst 4: Structural Design Pattern (MVC)

## Wat Ik Toon
Ik demonstreer dat ik **MVC (Model-View-Controller) pattern** correct toepas:
- **Model** - Data access layer (database queries)
- **View** - JSON responses (REST API)
- **Controller** - Business logic (request handling)
- **Routes** - URL routing (endpoint mapping)

**Bewijs:**
- Duidelijke folder structuur
- Scheiding van concernen
- Code georganiseerd per responsibility
- Volgt industry standards

## Hoe Dit In Mijn Project Zichtbaar Is
- **Models:** [models/](../../../models/) - Database operations
- **Controllers:** [controllers/](../../../controllers/) - Business logic
- **Routes:** [routes/](../../../routes/) - Endpoint definitions
- **Middleware:** [middleware/](../../../middleware/) - Cross-cutting concerns
